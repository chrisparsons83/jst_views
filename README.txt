Javascript Timer - Views Plugin
===============================

This is a module that adds views integration for the [Javascript Timer Module]
(http://drupal.org/project/jstimer).  This was initially developed for a
countdown clock on the [Broward County Florida Drupal Users Group]
(http://www.browarddrupal.org).

Current Status
--------------

This module is currently in development and is not suitable for use in
production websites in this state.
